Potater Discord bot 's API

Potater (the profile picture is a potato) bot has been a longtime project of mine which has gone through the traditional rollercoaster of making it and dedicating my time for a while, then forgetting about it, and remembering it once again. It started off at the same hackathon where I made LockdownLife, there was a panel speaker who taught us the very basics on how to make a discord bot. Fast forward almost four years later and I've upgraded Potater to include the latest slash commands and used python's request api in the process. Overall, I've been glad to pour time and effort into this bot, given how satisfying it feels to use the silly commands between me and my friends.
