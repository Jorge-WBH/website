DuckCard "UltraMerge" pandas & CustomTkinter

Over the summer of 2025, part of my work at the Stevens DuckCard office consisted of grabbing a snapshot of our school's credential database to determine what each student and staff's status is. Eventually, this information would be used to send a mass email to incoming students who needed to activate their IDs as well as updating a continous report to show trends.

This involved continous trial and error using Excel to quickly modify a formula that would parse through the database's parameters to determine every individual's status. To make this repeatable, I created a desktop GUI in Python using CustomTkinter and pandas for the interface and database processing respectively. The finished result provides an interface where you can select the two necessary CSVs and it would output a clear summary of the categories as well as filtering for the mass emails and reports.

Potater Discord bot 's API

Potater (the profile picture is a potato) bot has been a longtime project of mine which has gone through the traditional rollercoaster of making it and dedicating my time for a while, then forgetting about it, and remembering it once again.
